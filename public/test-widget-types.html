Test de Tipos de Widget AIPPS

Información

Esta página está diseñada para probar que el archivo embed.js funciona correctamente con ambos tipos de widget:

  • Bubble Widget: Botón flotante tradicional
  • Fullscreen Widget: Chat de pantalla completa estilo ChatGPT

Test 1: Widget Flotante (Bubble)

Este script debería crear un botón flotante en la esquina inferior derecha:

<script src="/embed.js?key=aipi_web_internal" data-widget-type="bubble"></script>

Test 2: Widget Pantalla Completa (Fullscreen)

Este script debería crear un botón que abre un chat de pantalla completa:

<script src="/embed.js?key=aipi_web_internal" data-widget-type="fullscreen"></script>

Test 3: Widget con Configuración Personalizada

Este script incluye configuración personalizada de color y posición:

<script src="/embed.js?key=aipi_web_internal" data-widget-type="bubble" data-theme-color="#ff5722" data-position="bottom-left" data-title="Asistente Personalizado"></script>

Logs de Consola

Abre las herramientas de desarrollador (F12) para ver los logs de la consola y verificar que:

  • El tipo de widget se detecta correctamente
  • Los parámetros de configuración se aplican
  • No hay errores de JavaScript
