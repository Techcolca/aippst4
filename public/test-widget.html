Mensajes Rotativos Implementados

Esta página está diseñada para probar el sistema de mensajes rotativos del widget AIPPS.

Características del sistema:
  • 7 mensajes comerciales diferentes
  • Rotación automática cada 7 segundos
  • Carga dinámica desde la base de datos
  • Mensajes de respaldo en caso de fallo del API

El widget flotante en la esquina inferior derecha debería mostrar diferentes mensajes cada 7 segundos.
