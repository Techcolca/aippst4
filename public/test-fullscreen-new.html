Prueba de AIPI - Widget de Pantalla Completa (Nueva Versión)

Esta es una página de prueba para el nuevo script de widget de pantalla completa de AIPI.

Instrucciones

Deberías ver un botón flotante en la esquina inferior derecha. Al hacer clic en él, se abrirá el chat en pantalla completa.

Características

  • Interfaz de usuario limpia y moderna
  • Experiencia de usuario mejorada
  • Transiciones suaves
  • Indicador de escritura durante la generación de respuestas
  • Diseño optimizado para móviles y escritorio

Contenido de ejemplo

Este texto está aquí para proporcionar contexto al chatbot. El widget escaneará esta página y utilizará su contenido para proporcionar respuestas relevantes.

AIPI es una plataforma conversacional potenciada por inteligencia artificial diseñada para integrarse en sitios web. Ofrece capacidades de chat en tiempo real y puede responder preguntas basadas en el contenido del sitio web.
